Interactive Pull-Chain Lamp 💡
A realistic, interactive web-based lamp project created using HTML, CSS, and JavaScript. This project simulates a desk lamp with a functional pull-chain mechanism, custom-shaped lamp shade, and smooth lighting transitions.
------------------------------------------------------------------------------------
🚀 Features:
Custom CSS Shapes: The lamp shade, stand, and base are all built using pure CSS (no images).

Interactive Pull-Chain: A realistic string that stretches when clicked (mousedown) and toggles the light when released (mouseup).

Smooth Transitions: Soft background and lighting transitions to mimic a real room environment.

Sound Effects: Included click sound for a more tactile user experience.
---------------------------------------------------------------------------------------
🛠️ Built With:
HTML5: Structural layout of the lamp components.

CSS3: Flexbox for alignment, z-index for layering, and transition for animations.

JavaScript: Event listeners to handle the pull-chain logic and state toggling.
---------------------------------------------------------------------------------------
🎮 How to Use:
Clone or download the repository.

Open index.html in any modern web browser.

Click and hold the handle at the end of the wire to pull it down.

Release the handle to toggle the lamp ON/OFF.
